### Cara Menggunakan Aplikasi:
- Tekan F11 untuk mode fullscreen.
- Untuk menutup mode fullscreen dapat 
  tekan Escape (Esc) atau F11 kembali.
- Untuk keluar dari Aplikasi dapat tekan Alt + F4
  atau keluar dari mode fullscreen lalu tekan ikon X
  di pojok kanan atas jendela Aplikasi.

### Penjelasan terkait menu/ option:
- Terdapat tombol option pada awal aplikasi dibuka,
  yang berada di pokok kanan atas.
- Target Db adalah besar suara yang ingin dicapai 
  oleh pemain.
- Toleransi Db adalah besar suara yang akan diabaikan
  oleh mic, biasanya digunakan untuk menghilangkan
  suara latar belakang.
- Timer Game adalah tenggat waktu pemain dalam bermain
  game itu sendiri.

### Mekanisme Game:
1. Akan ada Pertanyaan, lalu pemain harus tekan 
   tombol Start.
2. Pemain memasukan namanya pada kolom yang tersedia.
3. Jika pemain tidak memasukan namanya maka aplikasi
   akan memberikan notifikasi peringatan.
4. Setelah itu, Mohon persiapkan pemain karena dalam
   hitungan ke-3 timer game akan berjalan.
5. Jika Pemain berhasil mencapai target Db, maka layar
   akan berganti dengan ucapan selamat.
6. Jika Pemain gagal (timer game habis), maka layar
   akan berganti dengan ucapan coba lagi.
7. Terdapat Tombol Reset pada kedua layar diatas 
   (No. 6, 7), yang bertujuan untuk mereset game
   kembali pada taham awal (No. 1).
